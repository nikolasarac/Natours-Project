# Natours

This repository contains the final project for the **Advanced CSS & Sass** course by Jonas Schmedtmann. Natours is a fictional nature-tour landing page, built to showcase advanced CSS skills.

## Overview

Natours is a clean, modern landing site for a tour company focused on outdoor adventures. The design puts a strong emphasis on nature, featuring large images, animated buttons, and a responsive layout. The site uses advanced CSS concepts such as Sass, CSS Grid, media queries, BEM architecture, and CSS animations.

## Technologies Used

- HTML5  
- CSS3 / Sass  
- Responsive Design (Media Queries)  
- CSS Animations  

## Features

- **Responsive Layout:** Works beautifully on desktop, tablet, and mobile.  
- **Sass Architecture:** Uses Sass variables, mixins, nesting, and partials to keep styles modular and maintainable.  
- **BEM Naming Convention:** Clean and scalable class naming for easier styling and maintenance.  
- **Interactive Elements:** Animated buttons, hover effects, and smooth transitions.  
- **Grid & Flexbox Layouts:** CSS Grid for complex layouts, Flexbox for smaller components.  
- **Popup Modals:** Booking popups to simulate a real tour booking flow.
